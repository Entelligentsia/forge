# /forge:regenerate — Renamed in v1.0

This command was renamed to **`/forge:rebuild`** in v1.0.

```
/forge:regenerate                     → /forge:rebuild
/forge:regenerate workflows           → /forge:rebuild workflows
/forge:regenerate workflows plan_task → /forge:rebuild workflows plan_task
/forge:regenerate personas            → /forge:rebuild personas
/forge:regenerate skills              → /forge:rebuild skills
/forge:regenerate commands            → /forge:rebuild commands
/forge:regenerate templates           → /forge:rebuild templates
/forge:regenerate knowledge-base      → /forge:rebuild knowledge-base
/forge:regenerate tools               → /forge:rebuild tools
```

See [`/forge:rebuild`](rebuild.md) for full documentation.
